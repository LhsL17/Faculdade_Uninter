Plataforma web para alfabetização digital de pessoas leigas em tecnologia

O Conecta Fácil é uma plataforma educativa criada para orientar pessoas com pouca familiaridade com tecnologia no uso básico da internet, segurança digital, e-mail, aplicativos e boas práticas online. O objetivo é promover inclusão digital de forma prática, acessível e intuitiva.

Sobre o projeto

Esta solução foi desenvolvida para atender um grupo de pessoas com baixa alfabetização digital, especialmente adultos e idosos que enfrentam dificuldades no uso cotidiano da tecnologia. A proposta une educação e TI por meio de um site informativo, responsivo e acessível.

🎯

Objetivo principal

Ensinar noções básicas de tecnologia, navegação segura e uso de ferramentas digitais por meio de uma plataforma web intuitiva.

👥

Público-alvo

Pessoas leigas em tecnologia, com pouca experiência em internet, aplicativos, criação de senhas e prevenção contra golpes digitais.

🌐

Aplicação prática

O sistema pode ser apresentado em oficinas, grupos de capacitação, turmas de extensão ou atendimentos orientados de inclusão digital.

Metodologia

O projeto foi estruturado em etapas sequenciais, desde a identificação das dificuldades do público-alvo até a construção da plataforma e avaliação dos resultados.

1

Levantamento das dificuldades

Mapeamento das principais dúvidas do público leigo em relação ao uso de internet, senhas, e-mails e aplicativos.

2

Definição dos conteúdos

Seleção de temas prioritários, com linguagem simples, foco prático e organização por módulos de aprendizagem.

3

Modelagem da interface

Planejamento de um layout limpo, responsivo, intuitivo e acessível, adequado para celulares e computadores.

4

Desenvolvimento da aplicação

Construção do site com HTML, CSS e JavaScript, incluindo recursos de acessibilidade e quiz interativo.

5

Teste e validação

Verificação da navegação, legibilidade, responsividade e funcionamento dos recursos interativos.

6

Apresentação e avaliação

Disponibilização do sistema para demonstração prática e análise dos resultados esperados na inclusão digital.

Exemplo de lógica JavaScript aplicada

Abaixo está um trecho do script usado para calcular o resultado do quiz, demonstrando a parte prática do projeto em código. 

function corrigirQuiz() {
  const respostas = ['q1', 'q2', 'q3'];
  let pontuacao = 0;

  respostas.forEach((questao) => {
    const selecionada = document.querySelector(`input[name="${questao}"]:checked`);
    if (selecionada) {
      pontuacao += Number(selecionada.value);
    }
  });

  const resultado = document.getElementById('quizResult');
  resultado.classList.add('show');

  if (pontuacao === 3) {
    resultado.className = 'result show success';
    resultado.innerHTML = 'Excelente! Você acertou todas as questões.';
  } else {
    resultado.className = 'result show warning';
    resultado.innerHTML = `Você acertou ${pontuacao} de 3 questão(ões).`;
  }
}
